# Meta & AI Discovery

This page provides machine-readable resources and metadata about the ProductBay documentation. These are primarily intended for search engines, crawlers, and Large Language Models (LLMs).

## AI & LLM Integration

For AI assistants and Large Language Models, we provide consolidated text files containing the full documentation context. This allows models to answer technical questions about the plugin accurately.

* **[llms.txt](/productbay/llms.txt)**: Brief summary of the plugin and quick links.
* **[llms-full.txt](/productbay/llms-full.txt)**: The complete documentation compiled into a single file for easy AI consumption.

## Search Engines & Discovery

* **[Sitemap](/productbay/sitemap.xml)**: XML Sitemap containing all available URLs for the documentation.
* **[Robots.txt](/productbay/robots.txt)**: Crawler directives.
