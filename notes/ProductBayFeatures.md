# ProductBay Pro Features

This document tracks features planned for the Pro version of ProductBay.

## Custom Field Column Enhancements

### Advanced Meta Field Selector
- **Current (Free)**: Hybrid approach with quick-select common fields + manual text input
- **Pro Feature**: Searchable dropdown with all available product meta keys fetched from database
  - Auto-detect ACF, Meta Box, and other custom field plugin fields
  - Live preview of meta field values
  - Field type detection (text, number, date, etc.)
  - Formatted output based on field type
