### Each table column will have following as advanced settings:
- Show column heading (true/false)
- Column width 
  - Unit: auto (default, browser determines), px (fixed pixels), % (percentage)
  - Value: number (only applicable for px and %)
- Responsive visibility
  - all (show on all devices - default)
  - desktop (desktop only)
  - tablet (tablet only)
  - mobile (mobile only)
  - not-mobile (hide on mobile)
  - not-desktop (hide on desktop)
  - none (hidden completely)
- Order number (number)

### Custom Field (cf) Column Settings:
- metaKey: The product meta key to display (e.g., _weight, _length, custom_field)
- prefix: Text to show before the value (e.g., "Weight: ")
- suffix: Text to show after the value (e.g., " kg")
- fallback: Value to display if meta field is empty (e.g., "N/A")

### Combined Column Settings:
- layout: "inline" (horizontal) or "stacked" (vertical)
- elements: Array of column types to combine (minimum 2 required)
